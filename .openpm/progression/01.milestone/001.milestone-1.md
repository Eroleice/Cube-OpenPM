# 001.milestone-1

- 来源：DESIGN.md + MSC.md

## 目标

建立浏览器 3D 魔方程序的最小可见闭环：用户打开页面后能看到完成态 3D 魔方，能在 3 阶、4 阶、5 阶之间切换，并看到基础操作控件与计数器初始状态。

该 milestone 的核心不是只搭技术底座，而是交付一个可被用户观察和验证的页面初始体验，同时固化后续所有交互、打乱、求解都必须依赖的状态与动作公共边界。

## 范围

包含：

- 页面初始加载体验：
  - 默认显示 3 阶完成态魔方。
  - 显示“随机打乱”“求解”“速度”“3 阶”“4 阶”“5 阶”等基础控件。
  - 显示打乱步骤数和求解步骤数，初始为 0 或等价明确空状态。
- 3 阶、4 阶、5 阶完成态渲染：
  - 能按当前阶数展示对应结构的 3D 魔方。
  - 能表达各面颜色状态。
- 阶数切换基础闭环：
  - 用户选择 3/4/5 阶后，魔方重建为对应阶数完成态。
  - 切换后计数器重置为 0。
  - 切换后不保留上一阶魔方内部状态。
- 最小状态模型与公共契约：
  - 定义并使用 `CubeSize`，仅允许 3、4、5。
  - 建立 `CubeState` 完成态创建、完成态判断的基础能力。
  - 建立 `Move` / `MoveSequence` 的最小公共语义，为后续手动转动、打乱和求解共用。
  - 建立 `RunMode` / `PlaybackState` 的最小状态，至少覆盖 `idle` 与后续自动流程可扩展边界。
- 3D 渲染与逻辑状态边界：
  - `CubeState` 作为魔方逻辑状态唯一权威来源。
  - 渲染层只消费 `CubeSize` 和 `CubeState`，不自行成为业务状态源。
- 浏览器 3D 能力不可用时的基础提示或可诊断失败方式。

不包含：

- 不实现鼠标右键视角旋转。
- 不实现鼠标左键拖动层转动。
- 不实现随机打乱 20 步自动播放。
- 不实现自动求解。
- 不承诺“最优解”。
- 不引入账号、云存储、排行榜、教学、移动端触摸、主题皮肤或 3/4/5 阶以外魔方。
- 不要求完整性能优化，只要求 3/4/5 阶完成态基本可展示。

风险与依赖：

- 3D 渲染库、项目技术栈和构建入口需在实施前基于仓库真实现状确认；如需新增第三方库，应由 Architect 复核并记录用途、替代方案、包体/兼容性/维护风险。
- `CubeState` 表达方式、`Move` 语义和渲染动画协议属于后续破坏性变更高风险点，若实现阶段做出不可逆选择，应补充 ADR。
- 本 milestone 默认切换阶数后计数器重置为 0；若用户后续确认不同产品规则，需要调整后续任务。

## 预计 Task 切片

1. 页面骨架与 UI 控件初始状态  
   建立浏览器页面入口、基础布局、按钮/滑块/计数器/阶数切换控件，并保证初始文本状态可见。

2. 魔方状态模型与公共契约最小实现  
   建立 `CubeSize`、`CubeState`、完成态创建/判断、最小 `Move` / `MoveSequence` 类型边界和基础测试。

3. 3D 完成态渲染与渲染边界  
   基于当前 `CubeSize` 和 `CubeState` 渲染 3/4/5 阶完成态魔方，保证渲染层不直接拥有业务状态。

4. 阶数切换闭环  
   实现 3/4/5 阶切换、状态重建、计数器重置、旧视觉对象清理或替换，并覆盖 smoke/单元检查。

5. 初始错误与不支持反馈  
   对浏览器 3D 能力不可用、初始化失败等场景提供可理解反馈或可诊断错误路径。

## 验收

- 打开页面后，用户能看到一个 3D 魔方主体。
- 初始魔方处于 3 阶完成态，除非后续产品明确调整默认阶数。
- 页面显示：
  - 随机打乱按钮。
  - 求解按钮。
  - 打乱步骤数计数器。
  - 求解步骤数计数器。
  - 自动转动速度滚动条。
  - 3 阶、4 阶、5 阶切换入口。
- 初始打乱步骤数和求解步骤数显示为 0，或有等价明确空状态。
- 选择 3 阶后，画面显示 3 阶完成态魔方。
- 选择 4 阶后，画面显示 4 阶完成态魔方。
- 选择 5 阶后，画面显示 5 阶完成态魔方。
- 每次切换阶数后：
  - 魔方处于对应阶数完成态。
  - 打乱步骤数重置为 0。
  - 求解步骤数重置为 0。
  - 不继续展示或播放上一阶魔方状态。
- `CubeState` 不由 UI、输入模块或渲染模块直接改写。
- 相关状态模型、阶数合法性和完成态判断具备可测试证据。
- 用户可见初始页面具备 smoke 覆盖。
- 如引入第三方渲染库，必须说明依赖选择风险，并交由架构复核或记录 ADR 候选。
